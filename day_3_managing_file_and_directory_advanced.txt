1. what is permisson
    every file or dir has a Access Permission
    r : read
    w : write
    x : execute

    How to see the Access Permission
    code
        ls -l

    result
        -rw-r--r--  1 user user 1234 2ì›” 20 12:00 example.txt

    - : file, d : dir
    rw- : read, write, -
    r-- : group
    r-- : other

2. changing Access Permission

    code
        chmode +x myscript.sh
        chmode -x myscript.sh

3. changing owner
    code
        sudo chwon newuser myfile.txt
        sudo chwon newuser:newgroup myfile.txt

4. changing group
    code
        sudo chgrp newgroup myfile.txt

5. execute Permission
    code
        chmod +x script.sh
        ./script.sh

6. hard link
    code
        ln original.txt hard_link.txt

7. soft link
    code
        ln -s original.txt soft_link.txt
